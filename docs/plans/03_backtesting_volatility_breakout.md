# 3. 백테스팅 리뷰 (변동성돌파 관점, 상용화 중점)

작성일: 2026-01-02

---

## 1. 현재 백테스트 구조 요약
- next-open 체결 옵션 존재
- 슬리피지/수수료 기본 반영
- 전략/백테스트 분리 구조

---

## 2. 장점 (변동성돌파 관점)

- Look-ahead bias 방지 의식 있음
- 돌파 전략과 구조적으로 맞는 체결 가정
- 전략 비교/필터링 용도로는 유효

---

## 3. 치명적 한계 (상용 기준)

### (1) 시간축 왜곡
- entry/exit time이 현재시간 기준
- 보유기간/월별 성과 왜곡

### (2) 스탑/익절 체결 비현실적
- 봉 중 high/low 기반 스탑 미반영
- 변동성돌파의 핵심 리스크 과소평가

### (3) 거래비용 과소평가 가능성
- 슬리피지/체결 실패/부분체결 미흡

---

## 4. 리팩토링안 (최중요)

### (1) 체결 모델 고도화
- stop/TP는 candle high/low 기준
- intrabar 시뮬레이션 도입

### (2) 계좌 기반 리스크 평가
- % 합산 → equity curve 기반
- MDD, Calmar, Turnover 필수

### (3) 검증 프로토콜
- Walk-forward / Out-of-sample
- 파라미터 민감도 테스트

---

## 5. 결론
현재 백테스트는 **아이디어 검증용**.
상용 돌파 전략에는 **과도하게 낙관적**.
이 부분이 리팩토링 최우선.
